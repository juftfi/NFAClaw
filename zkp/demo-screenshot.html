ZK Proof for Verifiable Agent Transcript

Commit a transcript Merkle root on-chain per tokenId, then prove (in zero-knowledge) that a message hash is included in the committed set. Minimal, shareable skeleton for BAP-578-aligned agents.

What It Proves / What It Doesn't
be precise publicly
Proves: a specific transcript message hash is included in the committed set (root) for an agent/NFA.
Doesn't prove: that “training happened” or that a model was trained from that corpus.
If you want to prove “model output comes from a specific model version”, you need to commit model version + inference trace, which is a larger design.
